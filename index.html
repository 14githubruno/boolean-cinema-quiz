Tooltip

Since this is a boolean quiz, you will be able to submit only two answers: true or false. Here some shortcuts which can make your life much easier:
Ctrl + Alt + t for answering true
Ctrl + Alt + f for answering false
Ctrl + Alt + s for submitting the answer.

About

The quiz has a simple structure and a simple functionality too. There are three levels - basic, medium and advanced - and each of them stores ten assertions, which have to be proved only right or wrong, that is to say only true or false. Hence the name of the quiz contains boolean. The only topic is the world of cinema.
